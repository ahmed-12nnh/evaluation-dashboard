# دليلك لإطلاق موقع تقييم المتقدمين (النسخة الحديثة والفاخرة ✨)

تم تحويل الموقع بالكامل ليصبح موقعاً تفاعلياً حديثاً بتقنيات الويب النظيفة (بدون Streamlit). هذا يعني تصميماً أجمل، أسرع، ويمكنك استضافته مجاناً للأبد عبر GitHub Pages.

## الخطوة 1: تجهيز قاعدة البيانات في Supabase
صحيح أنك أنشأت قاعدة البيانات مسبقاً، لكن يجب أن ننشئ "الجدول" الخاص بالمتقدمين.
1. افتح مشروعك في [Supabase](https://supabase.com/dashboard/projects).
2. من القائمة الجانبية اليسرى، اضغط على **SQL Editor**.
3. اضغط على الزر **New Query**.
4. افتح ملف `setup.sql` الموجود على حاسوبك، انسخ كل الأكواد التي بداخله والصقها في تلك الصفحة.
5. اضغط على الزر الأخضر **Run** الموجود بالأسفل (أو في الأعلى). سترى رسالة نجاح.

---

## الخطوة 2: ربط الموقع بقاعدة البيانات
لكي يتمكن الموقع الجديد من قراءة الأسماء التي أدخلتها:
1. في Supabase، اذهب إلى إعدادات المشروع (أيقونة الترس **Settings**).
2. اختر قسم **API**.
3. ستجد هناك شيئين مهمين جداً:
   - **Project URL** (انسخه).
   - **Project API Keys**، ستجد مفتاح طويل مكتوب بجانبه (anon / public). قم بنسخه أيضاً.
4. الآن على حاسوبك الشخصي: افتح ملف `app.js` باستخدام تطبيق المفكرة (Notepad) أو أي محرر نصوص.
5. في السطر المكتوب فيه `const SUPABASE_URL`، استبدل الرابط برابط مشروعك.
6. في السطر المكتوب فيه `const SUPABASE_ANON_KEY`، استبدل النص الطويل بالمفتاح الذي نسخته للتو.
7. احفظ الملف (Ctrl + S).

---

## الخطوة 3: رفع الملفات وتشغيل الموقع
سنقوم برفع الملفات إلى نفس مستودع GitHub وسنفعل ميزة استضافة المواقع المجانية (GitHub Pages).
1. افتح مستودعك في GitHub (`evaluation-dashboard`).
2. اضغط على زر **Add file** ثم **Upload files**، وارفع الملفات الجديدة:
   - `index.html`
   - `style.css`
   - `app.js`
   ثم اضغط **Commit changes**.
3. الآن لإطلاق الموقع: من لوحة المستودع في الأعلى، اضغط على **Settings** (الإعدادات).
4. من القائمة الجانبية في صفحة الإعدادات، انزل قليلاً واختر **Pages**.
5. تحت قسم **Build and deployment**، ستجد خيار **Branch**.غيّر كلمة `none` إلى `main` واضغط على زر **Save**.

🎉 **مبروك!** 
انتظر دقيقة أو دقيقتين وقم بتحديث الصفحة، سيظهر لك رابط موقعك الجديد جاهزاً للاستخدام بالكامل في أعلى نفس صفحة الـ Pages. يمكنك فتح الموقع وإضافة متقدمين جدد من لوحة التحكم الخفية في الأسفل بالضغط على خيار (الإعدادات ⚙️) واستخدام الرقم السري الموقت "1234".
